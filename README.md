### Содержимое репозитория
Здесь содержатся для файла с программой.
- Один из них это блокнот в формате `.ipynb`. Удобство втом, что можно поочередно увидеть результат той или иной операции.
- Второй с расширение `.py`, это вся программа. Часть кода обернута в функции. После создания виртуальной среды, установки библиотек и создания базы данных, можно запускать данный файл.

После его запуска:
- Выполнится агрегация;
